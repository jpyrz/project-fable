# Project Fable celebration moments

Celebrations should confirm meaningful progress, not interrupt routine navigation. Use a major celebration for milestones and a smaller sparkle card for frequent actions.

## V1 celebration map

| Player moment | Level | V1 feedback |
| --- | --- | --- |
| Win or finish an arcade run | Major | Dedicated result scene, trophy animation, confetti, score, reward, replay |
| Craft an item | Major | Item reveal with its icon and confirmation that it reached the bag |
| Claim a daily task | Major | Gift reveal and exact Dewdrop reward |
| Make the daily wish | Major | Wish reveal and exact Dewdrop reward |
| Adopt the starter pet | Major | Planned onboarding welcome scene |
| Complete a collection set or earn a badge | Major | Planned milestone scene |
| Gain a reputation level | Major | Planned level-up scene with unlocked rewards |
| Buy an item | Minor | Treasure confirmation with the purchased item icon |
| Create a marketplace listing | Minor | Listing-live confirmation |
| Care for a pet | Minor | Pet-specific happy confirmation |
| Equip a cosmetic | Minor | Outfit confirmation with the item icon |
| Become friends | Minor | Planned friendship confirmation |

Errors never use celebratory styling. They remain visible until understood or retried. Motion respects the operating system's reduced-motion preference.
